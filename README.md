# automations

Central hub for managing work with Claude. See [`CLAUDE.md`](./CLAUDE.md) for how sessions are oriented.

## Quick start
- Drop raw inputs in `inbox/`, run `/inbox` to file them.
- `/triage` for Linear workload, `/brief <topic>` for a context pack, `/sessions` to recover context from prior Claude conversations.
- Integrations wired via MCP: Linear, Atlassian (Jira + Confluence), GitHub. M365 productivity MCP: TBD.
